# ZipBrute

Multi-Threaded ZIP files bruteforce tool made with Python 3 

### Installation & Usage:
`$ git clone https://github.com/CalfCrusher/ZipBrute`

`$ cd ZipBrute && pip install -r requirements.txt`

`$ python3 ZipBrute.py -f file.zip -d wordlist.txt`
